# Sistema de Cadastro de Clientes

Aplicação web simples para cadastro e gerenciamento de clientes, com integração a uma API externa.

## Funcionalidades
- Cadastro de clientes
- Listagem de clientes
- Exclusão de clientes

## Tecnologias
- JavaScript
- HTML
- CSS
- API (CrudCrud)

## API
Utiliza o CrudCrud para armazenamento dos dados.
- GET: listar clientes
- POST: cadastrar cliente
- DELETE: remover cliente

## Como usar
1. Clone o repositório:
git clone https://github.com/JacquelineTavaresDev/Sistema_de_Cadastro_de_Clientes
2. Abra o arquivo index.html no navegador

## Observação:
O CrudCrud gera URLs temporárias. Pode ser necessário atualizar a URL da API.
