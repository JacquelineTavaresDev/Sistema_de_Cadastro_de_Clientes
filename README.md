# Sistema de Cadastro de Clientes
## Descrição
Aplicação web simples para cadastro e listagem de clientes, desenvolvida com JavaScript, HTML e CSS.

## Funcionalidades
- Cadastro de clientes
- Listagem de clientes cadastrados

## Tecnologias
- HTML
- CSS
- JavaScript

## Como executar
1. Baixe o projeto
2. Abra o arquivo index.html no navegador
